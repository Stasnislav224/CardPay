
MiniBank — це міні-фінтех-проєкт на Java, який імітує базову роботу банківської системи.

Ключові можливості:

Створення користувачів та їх банківських акаунтів

Додавання та видалення кредитних карток

Поповнення та зняття коштів з картки

Переказ коштів між картками

Ведення логів транзакцій зі статусами (Pending, Completed, Failed)

Проєкт демонструє роботу з колекціями, об’єктами, інкапсуляцією, абстрактними класами та основами фінансової логіки.
